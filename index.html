Используемые технологии: C#, Windows Forms, Selenium Web Driver

Описание

Приложение, парсит случайные книги и создает коллекцию, которую затем можно экспортировать в файл JSON.

Работа с приложением:

  1. Установите значение количества книг (от 1 до 200).
  2. Установите ограничение на год выпуска. Книги, выпущенные ранее указанного года, будут пропущены.
  3. Нажмите кнопку "Запустить".
  4. Подождите, пока не появится окно с сообщением о завершении операции.
  5. Чтобы экспортировать полученную коллекцию, нажмите кнопку «Экспорт».
  6. Также в строке состояния вы можете увидеть информацию о работе парсера.

Парсинг выполняется асинхронно.

P.S. На данный момент приложение работает нестабильно. Иногда возможны ошибки, которые можно устранить, перезапустив приложение. В дальнейшем все проблемы будут устранены.
